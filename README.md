
# How to run

Data Visualization Dashboard with Interactive Charts
## Installation

Clone the repo
```bash
  git clone https://github.com/deepanshu112233/dashboard.git
```
Go to dashboard 
```bash
  cd dashboard
```
Run commands
```bash
    npm install react-scripts --save
    npm start
```
